## Installation

Install the project dependencies:

```bash
uv sync
```

Install the CLI globally in editable mode:

```bash
uv tool install -e .
```

To do both easier, use:
```bash
make install
```

Verify the installation:

```bash
cli-name --help
```

To uninstall:

```bash
uv tool uninstall cli-name
```
